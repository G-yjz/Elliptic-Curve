# Elliptic-Curve

对上面两条信息签名和验证：

A先对明文哈希，然后用自己的私钥对得到的哈希签名，把签名后的信息和A的公钥一起发送给B

B对密文解密得到明文，用A的公钥对明文验证，如果和A签名后的信息一样则返回1，否则返回0
